# Code of Conduct

## 🌍 Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone.

This project is focused on education in digital forensics and cybersecurity.
We expect all participants to act professionally and ethically.


## 🤝 Expected Behavior

Participants are expected to:

- Be respectful and constructive in communication
- Provide helpful feedback
- Focus on learning and collaboration
- Respect diversity of perspectives and experience levels
- Act ethically, especially given the sensitive nature of forensic topics


## 🚫 Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or offensive language
- Personal attacks or insults
- Sharing illegal, sensitive, or real-world confidential data
- Encouraging misuse of forensic techniques
- Any activity that violates laws or ethical standards


## ⚖️ Ethical Responsibility (Important)

Due to the nature of digital forensics:

- Do NOT promote illegal investigations
- Do NOT misuse tools or techniques from this project
- Do NOT attempt to apply labs directly to real-world cases without proper validation


## 🛠️ Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may:

- Remove or edit content
- Reject contributions
- Restrict participation

in response to violations.


## 📬 Reporting Issues

If you experience or witness unacceptable behavior:

- Open an issue (if appropriate), or
- Contact the maintainer privately

All reports will be handled with discretion.


## 📌 Scope

This Code of Conduct applies to:

- GitHub issues and pull requests
- Discussions and community interactions
- Any project-related communication


## ✅ Acknowledgment

By participating in this project, you agree to follow this Code of Conduct.

---

Thank you for helping build a responsible and ethical learning environment. 🚀
